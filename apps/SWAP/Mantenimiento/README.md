# Carpeta `includes`

Este directorio contiene los archivos y módulos esenciales para el funcionamiento del sistema de inventario y herramientas. Aquí se agrupan componentes reutilizables, scripts JavaScript, helpers PHP y servicios de backend.

## Estructura principal

- **components/**  
  Archivos HTML y CSS para encabezado, menú y pie de página.

- **db/**  
  Conexión a la base de datos.

- **helpers/**  
  Scripts PHP para consultas y lógica de negocio (módulos, herramientas, etc).

- **scripts/**  
  Archivos JavaScript para interacción, AJAX y lógica de frontend.

- **services/**  
  Servicios adicionales para el sistema.
