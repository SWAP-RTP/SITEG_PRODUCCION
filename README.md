1. Al hacer el git clone es importante que el archivo menu.js dentro de la carpeta de login , se cambie la ruta a: http://localhost:8086/, y en el archivo .gitignore agregar la ruta del menu , para que el archivo js no afecte al menu.js de produccion cuando se haga un push y se haga un merge de la tu rama con el main.
2. En el archivo logout.php ubicado en la carpeta auth dentro de login , igual cambiar la ruta y apuntar a la misma direccion:  http://localhost:8086/ para que todo se mantenga de manera local y no redirija a el servidor de produccion , y de igual manera agregar ak .gitignore para que al hacer un push no haya conflicto de rutas.
